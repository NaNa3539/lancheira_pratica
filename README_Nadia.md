# 🥗 Lancheira Prática

Este é um protótipo de sistema web desenvolvido para auxiliar pais e responsáveis na montagem de lancheiras escolares mais saudáveis, equilibradas e práticas para crianças e adolescentes do ensino fundamental.

## 🎯 Objetivo

Oferecer uma ferramenta educativa e acessível que:

- Sugere alimentos organizados por grupos (Frutas, Carboidratos, Proteínas, Lipídios, Hidratação)
- Calcula automaticamente o total de calorias
- Exibe alertas se:
  - A lancheira ultrapassar 400 kcal
  - Houver 2 ou mais alimentos ultraprocessados
- Elogia escolhas saudáveis (com alimentos in natura/minimamente processados abaixo de 400 kcal)

## 🛠️ Tecnologias utilizadas

- Node.js + Express
- HTML + CSS
- JavaScript
- VS Code

## ▶️ Como rodar o projeto localmente

1. Faça o download do repositório
2. No terminal, dentro da pasta do projeto, rode:

```
npm install
npm start
```

3. Acesse `http://localhost:3000` no seu navegador
